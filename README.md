# Infuser / EXIF Analysis


Infuser is a web-based EXIF Image Metadata Analysis Tool. 

  - EXIF Library
  - Django-based Infrastructure
  - Frontend & Backend

Information Provided by Image Analysis:
  - File Name
  - File Type
  - File Size
  - Manufacturer
  - Model
  - Image Height
  - Image Width
  - DateTime Original
  - DateTime Digitized
  - SubsecTime Digitized
  - Last Modified
  - GPS Info: Latitude
  - GPS Info: Longitude
  - Time Interval (GMT+)
  - Aperture Value
  - Brightness Value
  - Color Space
  - Offset
  - Exposure Bias Value
  - Exposure Mode
  - Exposure Program
  - Exposure Time
  - FNumber
  - Flash
  - Focal Length
  - Focal Length In 35mm Film
  - ISO Speed Ratings
  - Lens Make
  - Lens Model
  - Lens Specification
  - Metering Mode
  - Orientation
  - Resolution Unit
  - Scene Capture Type
  - Sensing Method
  - Shutter Speed Value
  - Software
  - Subject Location
  - White Balance
  - XResolution
  - YResolution
  - YCbCrPositioning
